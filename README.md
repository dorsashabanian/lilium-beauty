Lilium Beauty

A modern luxury beauty landing page inspired by the elegance of lily flowers.

## Features

- Fully Responsive Design
- Modern Hero Section
- CSS Grid Layout
- Flexbox Layout
- Product Showcase
- Newsletter Section
- Glassmorphism Elements
- Mobile Friendly Design

## Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid

## Assets Notice

To keep the repository lightweight and avoid large file uploads, image assets are not included in this GitHub version of the project.
The original project uses locally stored images inside an `assets` directory. 
In this repository, those images have been replaced with external image URLs to ensure the project remains fully functional and easy to preview.
The design, layout, responsiveness, and overall user experience remain identical to the original version.
